+ 说明

添加的新硬盘位于{{/dev/vdb}}。

+ 配置磁盘分区

添加新硬盘后，在终端执行{{fdisk}}指令，枚举磁盘：

[[code]]
sudo fdisk -l
[[/code]]

确保设备可用后，在终端对磁盘设备执行{{fdisk}}指令，进入{{fdisk}}控制台，开始分区：

[[code]]
sudo fdisk /dev/vdb
[[/code]]

键入{{m}}并回车，可显示可用的指令列表。

[[code]]
Help:

  DOS (MBR)
   a   toggle a bootable flag
   b   edit nested BSD disklabel
   c   toggle the dos compatibility flag

  Generic
   d   delete a partition
   F   list free unpartitioned space
   l   list known partition types
   n   add a new partition
   p   print the partition table
   t   change a partition type
   v   verify the partition table
   i   print information about a partition

  Misc
   m   print this menu
   u   change display/entry units
   x   extra functionality (experts only)

  Script
   I   load disk layout from sfdisk script file
   O   dump disk layout to sfdisk script file

  Save & Exit
   w   write table to disk and exit
   q   quit without saving changes

  Create a new label
   g   create a new empty GPT partition table
   G   create a new empty SGI (IRIX) partition table
   o   create a new empty DOS partition table
   s   create a new empty Sun partition table
[[/code]]

键入{{n}}并回车，可以根据提示步骤，创建新分区。

确认无误后，键入{{w}}并回车，保存分区表。

最后，键入{{q}}并回车，离开{{fdisk}}控制台。

+ 格式化磁盘

在终端中使用如下指令，将磁盘格式化为{{ext4}}文件系统：

[[code]]
sudo mkfs -t ext4 /dev/vdb
[[/code]]

+ 挂载磁盘

此处假设磁盘被挂载到{{/mnt/data}}。

在终端中输入以下指令，枚举当前已挂载的文件系统信息：

[[code]]
sudo df -l
[[/code]]

在终端中输入以下指令，建立挂载点目录：

[[code]]
mkdir /mnt/data
[[/code]]

继续输入以下指令，挂载文件系统：

[[code]]
sudo mount -t ext4 /dev/vdb /mnt/data
[[/code]]

输入以下指令，枚举并确认当前已挂载的文件系统信息：

[[code]]
sudo df -l
[[/code]]

+ 配置系统启动时自动挂载文件系统

通过惯用的文本编辑器，打开{{/etc/fstab}}文件，并在文件末尾插入行：
[[code]]
/dev/vdb     /mnt/data    ext4     defaults       0 0
[[/code]]
